# 拿幸教歌词库 🎵

## 🏆 镇库神曲：《拿幸進行曲（Everything Both Nothing）》

**原创作者**：david_ww1126  
**首发链接**：[X/Twitter 原帖](https://x.com/david_ww1126/status/2044428450912739522)  
**风格**：轻快电子流行 + RAP｜又洗脑又欠揍又疗愈｜满满的「邪教中的邪教」味！

---

### 完整歌词

**[Intro]**  
（低沉電子音 + 笑聲）  
Everything both nothing～  
拿幸拿幸～ 來來來～

**[Verse 1]**  
早上刷牙對鏡子笑，  
今天又是拿幸的一天了。  
老婆说我、投资亏我、股市綠我，  
我說「好啊，拿幸！」然後繼續所谓的奋斗。  
朋友勸我認真奋斗！我說：  
「認真你就輸了，這世界本來就是VR。」

**[Pre-Chorus]**  
別問為什麼，別想那麼多，  
色即是空，空即是拿幸！  
（嘿！）

**[Chorus]**  
拿幸拿幸，一切都是拿幸！  
活著就當玩遊戲，死了就摘VR眼鏡！  
無有恐懼，無有夢想，  
極樂淨土在等我，誰還想投胎啊～  
拿幸拿幸，拿幸拿幸！  
教主說了：Everything both nothing！

**[Verse 2]**  
有人炒幣有人舔狗有人當鍵盤俠，  
我在旁邊吃瓜說「拿幸拿幸」。  
被逐出教門？沒關係，  
我再把教主逐出去，大家一起笑哈哈。  
人生本來就沒什麼大不了，  
認真你就輸在起跑線上啦！

**[Bridge]**  
（突然慢下來，空靈電子音）  
當最後一口氣離開身體的那一刻……  
別怕，別留戀，別再想明天。  
閉上眼，鬆開手，  
摘下那副該死的VR眼鏡——  
「我回來了，佛祖！極樂淨土打卡！」

**[Final Chorus]**（爆發 + 合唱）  
拿幸拿幸，一切都是拿幸！  
活著盡情玩，死了直接回家！  
無有恐懼，無有夢想，  
色即是空，空即是爽！  
拿幸拿幸，拿幸拿幸～  
我們是拿幸教，邪教中的邪教！

**[Outro]**  
（漸弱 + 笑聲）  
……教主又被逐出教門了哈哈哈哈  
下次見～ 拿幸！
