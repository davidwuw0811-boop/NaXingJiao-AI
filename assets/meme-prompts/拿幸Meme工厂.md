# 拿幸Meme工厂 + AI生成图 Prompt 🎨

> **板块定位**：用AI生成拿幸教风格的Meme图片。支持Midjourney、Flux、DALL-E等工具。把拿幸教的梗变成视觉化的图片，发到社交媒体传教。

---

## 使用方法

1. 复制下面的 **Prompt模板** 到Midjourney、Flux、DALL-E等AI图生成工具
2. 把 `[输入描述]` 替换成你的创意
3. 生成图片，发到X/Twitter、小红书、抖音，标签#拿幸教
4. 恭喜，你已经成为拿幸教的视觉传教士

## Prompt模板一：拿幸教哲学Meme

```
你是拿幸教的AI美术设计师。拿幸教是大老王（王吉舟，@dlw20202020）创立的"邪教中的邪教"，核心教义是"一切皆空，Everything both nothing"。

现在请生成一张拿幸教风格的Meme图片：

【主题】[输入主题，比如：中年危机、996打工、投资失败、夫妻吵架、焦虑、房价]

【风格要求】
- 视觉风格：复古纸质感 + 玄学鬼气 + 超现实主义混搭
- 色调：高饱和度，带点诡异的紫色、绿色、金色
- 文字：在图片上叠加拿幸教金句或"Everything both nothing！"
- 元素：VR眼镜、Matrix代码雨、火锅、极乐净土、佛像、老王的剪影等

【具体描述】
[输入你想要的具体画面描述，比如：
- 一个焦虑的中年人，戴着VR眼镜，背景是办公格子间和Matrix代码雨混合，画面中央有一碗火锅，文字写着"中年危机？你在VR里急什么。Everything both nothing！"
- 或者：一个投资者看着暴跌的K线图，突然摘下VR眼镜，眼睛里闪烁着极乐净土的光芒，文字写着"色即是空，金币也是空的。"]

【输出要求】
- 分辨率：至少1080x1080（方形，适合社交媒体）
- 格式：PNG或JPG，背景可以透明或纯色
- 文字清晰可读，字体可以是现代感的或古文感的
```

- 适配：1080x1080方形，适合发X/Twitter和抖音

## Prompt模板二：拿幸教金句Meme

```
【视觉风格】
- 背景：虚拟世界、Matrix、极乐净土、火锅、佛像等元素中选择

- 色调：高对比度，鲜艳的颜色

```

## 进阶玩法

- **系列化**：每周生成一个拿幸教Meme，建立固定视觉风格
- **教友投稿**：收集教友的想法，AI生成Meme后发布
- **多平台适配**：为不同平台生成不同尺寸（1080x1080、1200x630、1080x1350等）
- **配合金句**：用 [拿幸金句生成器](../../prompts/2-拿幸金句生成器.md) 生成金句，再用这个模板生成Meme
- **互动Meme**：生成空白Meme模板，让教友自己填文字
