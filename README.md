## 🚀 快速开始

### 方式1：直接用Prompt（推荐）

1. 选择上面12个板块中的任意一个
2. 复制对应的Prompt模板
3. 粘贴到ChatGPT/Claude/Gemini等AI工具
4. 输入你的问题/新闻/投资烦恼等
5. 瞬间获得拿幸风格的回答

### 方式2：本地运行脚本

```bash
# 克隆仓库
git clone https://github.com/davidwuw0811-boop/NaXingJiao-AI.git
cd NaXingJiao-AI

# 运行每日挑战生成器
python3 scripts/每日挑战生成器.py
```

- Fork这个仓库

- 提交PR（记得加上"Everything both nothing！"）
