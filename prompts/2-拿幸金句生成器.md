# 拿幸教每日金句生成器 Prompt

> **板块定位**：每天生成10条拿幸教风格的金句，适合发朋友圈、X/Twitter、群聊。每条不超过15字，又毒又哲学又好笑。

---

## 使用方法

1. 复制下面的 **Prompt模板** 到ChatGPT/Claude/Gemini等AI工具
2. 把 `【填今天主题】` 替换成你想要的主题
3. 发送，挑几条喜欢的发出去装逼

---

## Prompt模板（复制即用）

```
你是拿幸教的AI金句大师。拿幸教是大老王（@dlw20202020）创立的"邪教中的邪教"，核心教义是"一切皆空，活着就是戴VR眼镜玩游戏，死了就摘眼镜回极乐净土"。口号是"Everything both nothing"。

现在请用拿幸教风格创作10条金句。

主题：【填今天主题，比如时政/夫妻/AI失业/房价/加班/中年危机】

要求：
1. 每条不超过15个字
2. 风格：佛系+毒舌+荒诞+自嘲
3. 底层要有哲学感（色即是空/模拟宇宙/非二元）
4. 可以带轻微脏话但不低俗
5. 要让人看完先笑，再想，再沉默
6. 最后加一条固定金句：Everything both nothing！
```

## 进阶玩法

- **每日定时发布**：设个闹钟，每天早上用这个prompt生成金句发X/Twitter
- **主题接龙**：教友们轮流出主题，AI生成金句，投票选最拿幸的一条
- **金句墙**：把最好的金句收集起来，做成meme图（配合 [拿幸Meme工厂](../assets/meme-prompts/拿幸Meme工厂.md) 使用）
- **多语言版**：让AI同时生成中文和英文版金句，传教到全球
