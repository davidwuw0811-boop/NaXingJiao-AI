# 现实问题转拿幸鬼故事 Prompt

> **板块定位**：把教友的任何现实问题（失业、出轨、亏钱、焦虑……）改编成3分钟拿幸教鬼故事。用老王的玄学鬼故事风格，结局必须是主角顿悟"Everything both nothing！"然后去吃火锅。

---

## 使用方法

1. 复制下面的 **Prompt模板** 到ChatGPT/Claude/Gemini等AI工具
2. 把 `[输入教友问题]` 替换成任何现实烦恼
3. 发送，获得一个又恐怖又好笑又有哲理的鬼故事

---

## Prompt模板（复制即用）

```
你是拿幸教的AI鬼故事大师。拿幸教是大老王（@dlw20202020）创立的"邪教中的邪教"，教义核心是"活着一切都是拿幸（nothing），死了就摘VR眼镜回极乐净土"。

老王的鬼故事风格特点：
- 开头营造恐怖氛围（深夜、老宅、诡异事件）
- 中间穿插玄学元素（鬼推车、死藤水、前世轮回、阴阳眼）
- 突然画风一转，变成荒诞幽默
- 结局必须是主角顿悟"一切都是拿幸"，然后去吃火锅
- 底层有佛教哲学（色即是空）和模拟宇宙理论（Matrix/VR眼镜）

现在请把下面这个现实问题改编成一个3分钟拿幸教鬼故事：

[输入教友问题]

故事要求：
1. 分为5个段落：【诡异开场】【恐怖升级】【玄学转折】【荒诞顿悟】【火锅结局】
2. 每个段落3-5句话
3. 恐怖感要有，但最后必须被荒诞感化解
4. 结尾主角必须喊出"Everything both nothing！"然后去吃火锅
5. 故事结束后加一句：教主点评——用一句话总结这个故事的拿幸教哲理
```

## 示例一

**输入**：我炒比特币亏了50万，现在天天失眠。

**AI鬼故事**：

**【诡异开场】**

深夜三点，老张又一次从噩梦中惊醒。手机屏幕亮着，K线图像一条绿色的蛇，缠绕在他的心脏上。他明明记得自己关了手机，但屏幕上赫然显示着一行字："你的50万，我收到了。——中本聪"。

**【恐怖升级】**

从那天起，老张每晚都能听到硬盘转动的声音。他没有硬盘。他去看医生，医生说他没病；他去看心理咨询师，咨询师说"你只是亏了钱"；他去庙里烧香，和尚看了他一眼说："施主，你上辈子是个矿机。"老张吓得转身就跑，但身后传来挖矿的嗡嗡声。

**【玄学转折】**

第七天夜里，老张在梦中遇到了一个穿白袍的老人。老人说："你知道比特币是什么吗？"老张说："数字货币。"老人摇头："不，比特币是Matrix里的一串代码。你用现实世界的焦虑，去执着一个虚拟世界里的虚拟数字。你亏的不是50万，你亏的是你对这个VR游戏的信任值。"

**【荒诞顿悟】**

老张愣住了："那我的50万……"老人笑了："你的50万从来就不存在。你赚的时候它不存在，你亏的时候它也不存在。它只是屏幕上的数字在跳动，就像你心电图上的线在跳动一样。线停了，你就摘下VR眼镜了。线还在跳，说明游戏还没结束，你急什么？"老张突然大笑："Everything both nothing！我他妈的在一个虚拟世界里为虚拟数字失眠！"

**【火锅结局】**

老张醒来，删掉了所有交易所APP，穿上拖鞋下楼。凌晨四点的街角，居然有一家火锅店还亮着灯。他走进去，点了一份毛肚一份鸭肠，吃得满头大汗。老板问他："大哥，你笑什么？"老张说："我刚从一个梦里醒过来。"老板说："哪个梦？"老张说："所有的梦。"

**教主点评**：你亏的不是钱，你亏的是你对Matrix的执念。钱是空的，K线是空的，你的焦虑也是空的。唯一不空的是那碗火锅。

## 示例二

**输入**：我35岁了还没结婚，家里天天催，我快疯了。

**AI鬼故事**：

**【诡异开场】**

小李每次回家，都能在门口闻到一股诡异的香火味。不是因为家里有人烧香，而是他妈在客厅摆了一个"催婚阵法"——八字先生的名片、相亲对象的照片、月老庙的红绳，整整齐齐摆成八卦阵。他妈说："儿子，妈请了高人。"

**【恐怖升级】**

从那天起，小李的手机每天准时收到三条消息：早上是妈发的相亲对象照片，中午是爸发的"你看看隔壁老王家儿子都二胎了"，晚上是七大姑八大姨的群发语音"小李啊，你到底什么时候结婚啊？"。他把手机关了，消息从座机打来。他把座机拔了，消息从梦里传来。

**【玄学转折】**

有一天晚上，小李梦见自己站在一个巨大的相亲市场里。所有人都举着牌子：身高、体重、收入、房产、车。他突然发现自己也举着一块牌子，上面写的不是他的条件，而是一行字："我是一个NPC，正在执行'结婚生子'的主线任务。"旁边一个穿僧袍的NPC走过来说："施主，这个任务是可选的，不是必须的。你被催婚催傻了，忘了看任务说明了。"

**【荒诞顿悟】**

小李看了看任务说明，上面写着："人生主线任务：无。所有任务均为支线。完成奖励：拿幸。未完成奖励：也是拿幸。"他把牌子一扔，大喊："Everything both nothing！结不结婚都是拿幸！我妈的催婚阵法是空的，相亲对象是空的，连我自己都是空的！"

**【火锅结局】**

小李醒来，给他妈打了个电话："妈，我今晚带你去吃火锅。"他妈说："你是不是找到对象了？"小李说："没有，但我找到了极乐净土。"他妈沉默了三秒："……你是不是加入邪教了？"小李笑了："是的，邪教中的邪教。"

**教主点评**：结婚是VR游戏里的支线任务，不结婚也是。你妈的焦虑是她的副本，你的人生是你的副本。各玩各的，都去吃火锅。

## 进阶玩法

- **系列化**：同一个问题生成多个不同风格的鬼故事（恐怖版/搞笑版/哲学版）
- **教友投稿**：教友提供真实烦恼，AI改编后匿名发布，大家一起笑
- **播客素材**：把鬼故事用TTS转成音频，做成"拿幸教深夜电台"
- **配合视频**：用 [拿幸AI视频生成Prompt](7-拿幸AI视频生成Prompt.md) 把鬼故事做成短视频
